## Donwload

Clone Project

```bash
  git clone https://github.com/thquan105/Bookshop.git
```

-   Copy .env.example into .env then edit the database and api key
- Install composer
```bash
    composer install
```
- Create Database
```bash
    php artisan artisan migrate:fresh --seed
```
- Create link storage
```bash
    php artisan storage:link
```
- Serving Laravel

```bash
    php artisan serve
```

```bash
    npm run dev
```
